## 1. T3MP3ST — 自動化紅隊測試平台

**倉庫：** [elder-plinius/T3MP3ST](https://github.com/elder-plinius/T3MP3ST)  
**⭐ Stars：** 4,020 | **語言：** TypeScript

### 項目簡介

T3MP3ST 是一個針對 AI 系統的自主紅隊（Red Team）攻擊測試平台。它採用多代理（multi-agent）架構，能夠自動化地對目標 AI 系統發起安全測試，模擬真實攻擊場景，幫助開發者提前發現模型的安全漏洞。

### 為何火紅

隨著 AI 模型在生產環境中的廣泛部署，安全性成為最關鍵的議題之一。T3MP3ST 提供了一套完整的自動化安全評估框架，在開源社群中引發了廣泛討論。

### 技術棧

- **TypeScript** — 主要開發語言
- **多代理架構** — 模擬多個攻擊者協同作業
- **模組化插件系統** — 可擴展攻擊向量

### 應用場景

- AI 模型上線前的安全審計
- LLM 紅隊測試自動化
- 安全研究人員的評估工具

## 3. OpenScience — 開源 AI 科學研究平台

**倉庫：** [synthetic-sciences/openscience](https://github.com/synthetic-sciences/openscience)  
**⭐ Stars：** 1,865 | **語言：** TypeScript

### 項目簡介

OpenScience 是一個專為科學研究設計的開源 AI 工作台。它整合了數據分析、模型訓練、論文管理等功能，旨在讓科學家能夠在一個統一的平台上完成從數據探索到論文發表的完整流程。

### 為何火紅

AI 正在深刻改變科學研究的方式。OpenScience 將最先進的 AI 工具與學術工作流程結合，降低了科研人員使用機器學習的門檻。它代表了「AI for Science」這一重要趨勢。

### 技術棧

- **TypeScript** — 前端與後端
- **模塊化架構** — 支持插件擴展
- **API-first 設計** — 易於集成

### 應用場景

- 學術研究團隊的協作平台
- 跨學科的數據分析工作
- 論文實驗的可重複性管理

## 4. investing-for-beginners — 投資百科知識框架

**倉庫：** [xuchonglang/investing-for-beginners](https://github.com/xuchonglang/investing-for-beginners)  
**⭐ Stars：** 1,537 | **語言：** 文檔

### 項目簡介

「小隱寺投資百科」是一個面向初學者的投資知識框架，覆蓋美股、期權與加密貨幣等領域。它以結構化的方式組織投資知識，幫助新手建立系統化的投資認知。

### 為何火紅

個人投資理財在華語社群中持續升溫。這個項目以開源方式整理了一套完整的投資知識體系，滿足了大量初學者「從零開始學投資」的需求。

### 內容涵蓋

- **美股投資** — 基礎知識到策略分析
- **期權交易** — 概念解讀與風險管理
- **加密貨幣** — 區塊鏈技術與投資方法

### 應用場景

- 投資新手入門學習
- 投資知識體系搭建
- 作為個人研究的參考資料庫

## 5. Knockoff — Amazon 購物淨化器

**倉庫：** [Shpigford/knockoff](https://github.com/Shpigford/knockoff)  
**⭐ Stars：** 1,472 | **語言：** JavaScript

### 項目簡介

Knockoff 是一個 Chrome 瀏覽器擴展，它能夠自動過濾 Amazon 搜尋結果中的山寨品牌和低品質商品，讓用戶只看見真實的、有歷史的知名品牌產品。

### 為何火紅

在 Amazon 上購物時，搜尋結果經常被各種不知名的山寨品牌淹沒。Knockoff 以極簡的方式解決了這個普遍痛點——安裝後無需配置，自動生效。其名稱本身也極具傳播力。

### 技術棧

- **JavaScript** — Chrome Extension Manifest V3
- **輕量級** — 約束 API 權限，保護用戶隱私

### 應用場景

- Amazon 日常購物體驗提升
- 避免購買山寨/白牌產品
- 電商平台用戶體驗研究
